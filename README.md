# Challenge Alura Store LATAM – Proyecto de Data Science

Este proyecto analiza los datos de ventas de las cuatro tiendas de **Alura Store** para ayudar al Sr. Juan a decidir cuál vender.  
Se trabajó con **Python (Pandas y Matplotlib)**, realizando análisis y gráficos sobre:

- Ingreso total por tienda  
- Ventas por categoría  
- Valoración promedio de los clientes  
- Productos más y menos vendidos  
- Costo de envío promedio
- Gráficos Adicional:
  * Evolución mensual de ingresos por tienda
  * Relación de costo de envío y calificación

## Herramientas utilizadas
- Python 3  
- Google Colab  
- Pandas  
- Matplotlib  

## Resultados
- La **Tienda 4** presenta el menor nivel de facturación y calificación.  
- Tiene también el **mayor costo de envío promedio**, lo que la hace menos competitiva.  
- Se recomienda su venta, priorizando las demás por su mayor estabilidad y satisfacción del cliente.

## Cómo ejecutar el proyecto
1. Descarga el archivo `.ipynb` o abre directamente en Google Colab.  
2. Ejecuta cada celda en orden para reproducir el análisis.
